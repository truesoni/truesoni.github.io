EIRA- An AI GuideBot

  • Tech Stack: Python, PyTorch, NumPy, NLTK, Pyttsx3, Speech Recognition, Tkinter, pyglet, and JSON
  • Technical Papers: Link

EIRA - An AI GuideBot, is a software that uses its Artificial Intelligence (AI) in order to do interactions with humans. The program is intended to solve the queries as fast as possible with minimal delay and give back their respective solutions and in case of questions, their respective answers.

The core technology and core working principles behind the project, as well as their applications in multidimensional contexts such as healthcare, call centers, e-commerce, educational institutions, etc., are presented in this report.

The whole of this project was developed using Visual Studio Code (IDE), and by using Python language for programming the same.
